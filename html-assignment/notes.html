Horizontal Line <hr> tag:
It is used to create a break in the form of a horizontal line on a page. It is a self closing tag
Description List <dl> tag:
It is used to create a list of descriptions. It's closing tag is </dl>
Description Term <dt> tag:
This is one of the tags found within a <dl> (Description list) tag. It is used to state a term that we want to describe. It's closing tag is </dd>
Description Definition <dd> tag:
This is the second tag found within a <dl> (Description list) tag. It is used to describe a term that has been listed in a Description list. It's closing tag is </dd>.
Unordered List <ul> tag:
This tag is used when we want to create or make a list of items that are not in any particular order. It's closing tag is </ul>
